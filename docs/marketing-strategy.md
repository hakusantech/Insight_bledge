# InsightBridge マーケティング戦略

## 1. エグゼクティブサマリー

InsightBridge（インサイトブリッジ）は、経営人材の評価（アセスメント）とエグゼクティブコーチングに特化したプロフェッショナルサービス企業である。ラッセル・レイノルズ・アソシエイツ元日本代表の岩本秀一氏が2022年5月に設立。経営者層の「成長の孤立」という課題に対し、科学的アプローチで解決策を提供する。

本戦略では、InsightBridgeのブランド認知拡大、リード獲得、クライアント基盤の拡大を実現するための包括的なマーケティングプランを提示する。

---

## 2. 市場分析

### 2.1 市場環境

| 要素 | 内容 |
|------|------|
| 市場規模 | 日本のエグゼクティブコーチング市場は年間約300〜500億円規模（推定） |
| 成長率 | 年率10〜15%で成長中（企業のガバナンス改革・後継者育成ニーズの高まり） |
| 市場トレンド | VUCA時代の経営力強化、コーポレートガバナンス・コード改定、人的資本経営の推進 |

### 2.2 競合分析

| 競合カテゴリ | 代表例 | 差別化ポイント |
|-------------|--------|--------------|
| グローバルファーム | エゴンゼンダー、スペンサースチュアート | InsightBridgeは日本企業の文化・慣習に深く精通 |
| 国内コーチングファーム | コーチ・エィ、ビジネスコーチ | アセスメント×コーチングの一体提供が強み |
| 独立系コンサルタント | 個人コーチ、フリーランス | 心理学博士との連携による科学的アプローチ |

### 2.3 SWOT分析

| | ポジティブ | ネガティブ |
|---|-----------|-----------|
| **内部** | **強み**: グローバルサーチファーム20年の経験、心理学的アセスメントの専門性、経営層ネットワーク | **弱み**: 設立間もないブランド認知の不足、少人数体制 |
| **外部** | **機会**: ESG/人的資本開示の義務化、後継者計画の需要増、DX推進による経営変革 | **脅威**: 大手ファームの市場支配、景気変動による研修予算削減 |

## 3. ターゲットセグメンテーション

### 3.1 プライマリターゲット

**大企業の経営層・取締役会**

- 売上500億円以上の上場企業
- CEO後継者計画を検討中の企業
- ガバナンス改革に取り組む企業
- 意思決定者：CEO、CHRO、社外取締役

### 3.2 セカンダリターゲット

**人事・タレントマネジメント責任者**

- 次世代リーダー育成プログラムを推進する企業
- 幹部登用の精度向上を求める企業
- 意思決定者：人事部長、人材開発部長

### 3.3 ターシャリターゲット

**外資系企業の日本法人**

- 日本固有の経営課題を抱える外資系企業
- グローバル本社との橋渡しが必要な企業
- 意思決定者：日本法人代表、リージョナルHR

## 4. ブランドポジショニング

### 4.1 ポジショニングステートメント

> 「経営者の可能性を科学的に引き出す、唯一無二のパートナー」

InsightBridgeは、グローバルサーチファームでの実務経験と心理学的知見を融合し、日本の経営人材が自己の強みと課題を客観的に認識し、持続的な成長を実現するための支援を提供する。

### 4.2 バリュープロポジション

1. **経験の深さ** — 20年超のグローバルエグゼクティブサーチ経験に基づく実践知
2. **科学的アプローチ** — 心理学博士との連携によるエビデンスベースのアセスメント
3. **一気通貫の支援** — 評価から育成まで一体的に提供する独自モデル
4. **経営者視点** — 経営の現場を熟知したコンサルタントによる伴走型支援

## 5. マーケティング戦略

### 5.1 コンテンツマーケティング（最重要施策）

#### ソートリーダーシップの確立

| 施策 | 内容 | 頻度 | KPI |
|------|------|------|-----|
| ブログ/コラム | 経営人材育成・アセスメントに関する専門記事 | 月2本 | PV数、滞在時間 |
| ホワイトペーパー | 「日本企業のCEO後継者計画の実態と課題」等 | 四半期1本 | DL数、リード獲得数 |
| 事例紹介（匿名） | クライアント成功事例の詳細ケーススタディ | 月1本 | 問い合わせ転換率 |
| 代表メッセージ動画 | 岩本氏による経営者向け短尺動画コンテンツ | 月1本 | 視聴回数、シェア数 |

#### SEO戦略

ターゲットキーワード：
- 「エグゼクティブコーチング」「経営人材 アセスメント」（メインKW）
- 「CEO 後継者計画」「次世代リーダー 育成」（ロングテールKW）
- 「経営者 コーチング 効果」「幹部 評価 方法」（情報収集系KW）

### 5.2 リファラル＆ネットワーク戦略

エグゼクティブ向けサービスの特性上、紹介・口コミが最重要チャネルとなる。

| 施策 | 内容 |
|------|------|
| エグゼクティブ・ラウンドテーブル | 経営者10名限定の非公開ディスカッション（四半期開催） |
| アラムナイ・ネットワーク | コーチング修了者のコミュニティ構築・定期交流会 |
| パートナー紹介プログラム | 弁護士事務所・会計事務所・PEファンドとの連携 |
| ボードメンバー向け勉強会 | 社外取締役向け「後継者評価の実務」セミナー |

### 5.3 デジタルマーケティング

| チャネル | 戦略 | 優先度 |
|----------|------|--------|
| LinkedIn | 岩本氏個人アカウントでの発信＋企業ページ運用 | 高 |
| メールマガジン | 月次の経営人材インサイトレター（ターゲットリスト向け） | 高 |
| Google広告 | 検索広告（「エグゼクティブコーチング」等のKW）| 中 |
| note/はてなブログ | SEO流入を狙った長文記事プラットフォーム | 中 |
| X (Twitter) | 業界ニュースへのコメント、イベント告知 | 低 |

### 5.4 PR・メディア戦略

| 施策 | 内容 |
|------|------|
| 経営誌への寄稿 | 「Harvard Business Review 日本版」「DIAMOND ハーバード・ビジネス・レビュー」等 |
| 書籍出版 | 「日本の経営者を変える — アセスメント×コーチングの新潮流」（仮題） |
| カンファレンス登壇 | HRサミット、日本CHO協会、日本コーチング学会等 |
| メディア取材対応 | 日経ビジネス、東洋経済等のビジネスメディア |

### 5.5 ウェブサイト改善施策

現行サイト（Next.js）をベースに以下を追加：

| 施策 | 目的 |
|------|------|
| リード獲得フォーム | ホワイトペーパーDL・無料相談申込の導線強化 |
| チャットボット導入 | 初期問い合わせのハードルを下げる |
| 英語版コンテンツ拡充 | 外資系企業・グローバル案件への対応 |
| ブログセクション追加 | コンテンツマーケティングの基盤構築 |
| Google Analytics 4 + タグマネージャー | データドリブンな改善サイクルの確立 |

## 6. カスタマージャーニーとファネル設計

```
認知（Awareness）
  │  ソートリーダーシップ記事、LinkedIn投稿、カンファレンス登壇
  ▼
興味（Interest）
  │  ホワイトペーパーDL、メルマガ登録、セミナー参加
  ▼
検討（Consideration）
  │  事例紹介閲覧、無料相談申込、紹介による接触
  ▼
決定（Decision）
  │  提案書提出、パイロットプログラム実施
  ▼
契約（Purchase）
  │  アセスメント/コーチング契約
  ▼
推奨（Advocacy）
     アラムナイネットワーク参加、紹介、推薦コメント提供
```

## 10. リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| コンテンツ制作リソース不足 | 高 | 外部ライター・動画制作チームとの提携 |
| 景気後退による予算削減 | 中 | ROI訴求の強化、小規模パイロット提案 |
| 競合大手の価格攻勢 | 中 | 品質・専門性での差別化を徹底 |
| 個人ブランド依存リスク | 高 | パートナーの発信力強化、組織ブランドの構築 |
